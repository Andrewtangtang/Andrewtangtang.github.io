# Yun-Tang Chang — Portfolio

Personal portfolio and technical writing site for Yun-Tang (Andrew) Chang.

Live site: [andrewtangtang.github.io](https://andrewtangtang.github.io/)

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm test
```

The site is statically exported to `dist/client` and deployed to GitHub Pages by GitHub Actions.
